MindCare AI

MindCare AI is an AI-powered emotional wellness web application that helps users reflect on their feelings and receive empathetic, structured insights generated by AI.
This project focuses on emotional support and self-reflection, not medical or diagnostic advice.


Live Links
----------
Frontend:
https://mindcare-ai-vuhz.onrender.com

Backend API Docs:
https://mindcare-ai-vuhz.onrender.com/docs


Tech Stack
----------
Frontend:
- React (Vite)
- TypeScript

Backend:
- FastAPI (Python)
- Pydantic
- pydantic-ai
- MongoDB

Deployment:
- Render


Project Structure
-----------------

frontend/
  src/
  index.html
  package.json
  vite.config.ts


Environment Variables
---------------------
Set these in your deployment environment (do NOT commit .env):

OPENAI_API_KEY=your_openrouter_api_key
MONGO_URI=your_mongodb_connection_string


Setup and Run Locally
--------------------

Backend:
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

Backend runs at:
http://127.0.0.1:8000


Frontend:
cd frontend
npm install
npm run dev

Frontend runs at:
http://localhost:5173
